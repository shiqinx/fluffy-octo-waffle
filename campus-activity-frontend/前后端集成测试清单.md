## 🚀 启动后端服务

### 方式1：直接启动
```bash
cd backend
npm start
# 或
java -jar target/campus-activity-backend.jar
```

### 方式2：使用脚本
```bash
# Windows
switch-to-backend.bat

# Linux/Mac
./switch-to-backend.sh
```

## 📊 测试结果解读

### 成功指标 ✅
- 后端连接状态：200 OK
- API响应：正确数据格式
- 认证流程：token正常获取
- 数据操作：CRUD成功

### 失败处理 ❌
- 连接失败：检查后端服务
- 401错误：检查认证配置
- 404错误：检查API路径
- 500错误：检查后端代码

## 🔄 测试模式切换

### 切换到纯前端测试
```bash
# 修改 .env.development
VITE_USE_MOCK=true

# 重启前端服务
npm run dev
```

### 切换到集成测试
```bash
# 修改 .env.development
VITE_USE_MOCK=false

# 确保后端服务运行
# 重启前端服务
npm run dev
```
